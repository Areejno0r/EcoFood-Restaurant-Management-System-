# EcoFood –  Restaurant Management System

Designed and developed the backend of a full-featured healthy restaurant web system using Python and Django. Built core functionalities including online table reservations, order management, ingredient customization, and real-time order status tracking. Designed system architecture and database models (MySQL), and applied design patterns to ensure scalability and maintainability. Implemented multi-role system behavior (Customer, Manager, Chef, Receptionist, Delivery Driver, Admin) with role-based logic and workflows. Developed features such as payment handling, order tracking, reservation management, and reporting system. Collaborated with team members to organize tasks, define system requirements, and ensure smooth development workflow.

## Backend (Django)
- REST API with role-based workflows
- MySQL database with design patterns
- Order tracking, reservations, payments

## Frontend (React)
- React components for interactive user interfaces
- User dashboards for all 6 roles
- Real-time order status updates
- API integration with Django backend

## Team & Methodology
- Task management & requirements definition
- Collaborative development workflow
